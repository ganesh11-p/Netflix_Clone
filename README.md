# Netflix Clone 🎬

🔗 Live Demo: https://your-netlify-or-vercel-link.vercel.app

A fully responsive Netflix Clone built using HTML, CSS, and JavaScript.

This project recreates the modern Netflix landing page UI with smooth scrolling movie sliders, responsive layouts, FAQ accordion, glowing curved hero section, and interactive components.

---

# 🚀 Features

- Responsive Netflix-inspired UI
- Modern Hero Section
- Curved glowing border effect
- Trending Movies Slider
- Left & Right Scroll Buttons
- Smooth Horizontal Scrolling
- FAQ Accordion Section
- Footer Section
- Hover Animations
- Mobile Responsive Design
- Clean UI & Layout

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
